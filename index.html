Пустота

Будиночок для двох
Проживання в одномісному або двомісному будиночку з відокремленою територією. Кожний будиночок оснащений спальнею (або двома для двомісного варіанту) з окремими місцями для медитації, просторою вітальнею з зоною для йоги, власною кухнею та столом. Також кожний будиночок має терасу для відпочинку або занять. Проживання з партнером під час ретриту буде комфортним завдяки наявності двох спалень та зручних умов для занять і відпочинку.
